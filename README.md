## 📖 What This Is

This is a fully interactive, browser-based summer reading program for Pine Forge Academy Parent's Association high school students. It includes:

- A **master book list** for all four grade levels
- Four **SAT-prep reading journals** — one per grade — with writing prompts, vocabulary trackers, practice questions, and print/export functionality
- A **home page** that links everything together

Every prompt, vocabulary exercise, and practice question is aligned to a specific SAT reading or writing skill — so students build real test-prep skills while engaging with great literature.

## 🎯 SAT Skills Built Into Every Journal

| Skill | Description | Grades |
|-------|-------------|--------|
| **Claim · Evidence · Explanation** | Every prompt requires quoting the text and explaining the connection | All |
| **Vocabulary in Context** | Word tracker with part of speech, context definition, and original sentence | All |
| **Point of View & Narration** | How narrator choice shapes meaning — a top SAT question type | 9–11 |
| **Rhetoric & Persuasion** | Identify and evaluate persuasion techniques in speeches and arguments | 10–12 |
| **Argument Analysis** | Identify claim, evidence, reasoning quality, and counterargument | 10–12 |
| **Allegory & Symbol** | Track how images and events carry meaning beyond their literal function | 9–12 |
| **Motif Tracking** | Follow a recurring image and show how its meaning evolves | 11–12 |
| **Paired Text Synthesis** | Compare two works — the SAT's highest-difficulty reading skill | 9–12 |
| **Same-Author Synthesis** | Compare two works by the same author to reveal consistent beliefs | 12 |
| **Analytical Essay Writing** | Builds from paragraphs (9th) to full 4-paragraph college essays (12th) | All |

## 🎓 How Students Use the Journals

1. **Open** their grade's journal from the home page or nav bar
2. **Enter** their name and school on the cover
3. **Read** their required books, then choose from the choice options
4. **Answer** the SAT-aligned prompts as they read
5. **Build** their vocabulary list — word, part of speech, definition, original sentence
6. **Try** the SAT practice question for each required book (auto-graded)
7. **Print or Export** their completed journal to turn in

> Journal entries are saved automatically to the student's browser. Each student's responses are private to their own device.

---

## 🖨️ Turning In Work

Each journal has two submission options:

- **🖨 Print / Save PDF** — opens the browser print dialog; student saves as PDF
- **⬇ Export as Text** — downloads a `.txt` file with all responses neatly formatted
