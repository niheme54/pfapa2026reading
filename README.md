## 📖 What This Is

This is a fully interactive, browser-based summer reading program for Pine Forge Academy high school students. It includes:

- A **master book list** for all four grade levels
- Four **SAT-prep reading journals** — one per grade — with writing prompts, vocabulary trackers, practice questions, and print/export functionality
- A **home page** that links everything together

Every prompt, vocabulary exercise, and practice question is aligned to a specific SAT reading or writing skill — so students build real test-prep skills while engaging with great literature.

## 🎯 SAT Skills Built Into Every Journal

| Skill | Description |
|-------|-------------|
| **Claim · Evidence · Explanation** | Every prompt requires students to quote the text and explain the connection |
| **Vocabulary in Context** | Word tracker with part of speech, context definition, and original sentence |
| **Point of View & Narration** | How narrator choice shapes meaning — a top SAT question type |
| **Argument Analysis** | Identify claim, evidence, reasoning quality, and counterargument |
| **Paired Text Synthesis** | Compare two books — the SAT's highest-difficulty reading skill |
| **Analytical Essay Writing** | Builds from paragraphs (9th) to full 4-paragraph essays (12th) |

## 🎓 How Students Use the Journals

1. **Open** their grade's journal from the home page or nav bar
2. **Enter** their name and school on the cover
3. **Read** the books and answer the journal prompts as they go
4. **Build** their vocabulary list — word, part of speech, definition, sentence
5. **Try** the SAT practice question for each book (auto-graded with explanation)
6. **Print or Export** their completed journal to turn in to their teacher

> Journal entries are saved automatically to the student's browser. Each student's responses are private to their own device.

---

## 🖨️ Turning In Work

Each journal has two submission options at the bottom of every page:

- **🖨 Print / Save PDF** — opens the browser print dialog; student saves as PDF and submits
- **⬇ Export as Text** — downloads a `.txt` file with all responses formatted for easy reading
